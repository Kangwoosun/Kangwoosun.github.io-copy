# 1. Function call by string

```php
'phpinfo'(); // call phpinfo
```


# 2. Array indexing

```php
$array = array("a", "b", "c")

echo $array[1] // result : b
echo $array{1} // result : b
```


# 3. String operations

```php
'qerialize'|'r' // result : 'serialize'
```


# 4. Error by finfo()

```php
echo new finfo(0, '/')
```


# 5. Change variable to string

```php
qerialize|r($a) // result : print some error message and call serialize($a)
```
